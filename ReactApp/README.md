# Restaurant POS Frontend

This is the React frontend for the Restaurant POS system.

## Technologies
- React 18
- TypeScript
- Vite
- Tailwind CSS
- Zustand

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
